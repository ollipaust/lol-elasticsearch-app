## Features

* UI-Kit by appbase.io
* Elasticsearch
* Styled-Components

* Desktop-only! Not responsive!

## Installation

* `git clone https://github.com/ollipaust/lol-elasticsearch-app.git lolapp`
* `cd lolapp`
* `yarn install`
* `yarn start`
* visit http://localhost:3000/
