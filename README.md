## Live Sandbox



## Features

* UI-Kit by appbase.io
* Elasticsearch
* Craco for import aliases
* Styled-Components

## Installation

* `git clone https://github.com/ollipaust/lol-elasticsearch-app.git lolapp`
* `cd lolapp`
* `yarn install`
* `yarn start`
* visit http://localhost:3000/
